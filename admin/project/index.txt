==================
Project management
==================

in the project tree, the Application builder page will look as follows:

.. image:: /admin/_images/project_node_jampy.png
	:scale: 50%
	:align: center
	:alt: Project management


.. important::

        ``Jam.py`` framework ``version`` number.
